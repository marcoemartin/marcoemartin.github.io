Creating My First Hugo Website

September 2, 2017   

For some time now I have been wanting to create a personal website to show case my projects and as a way to let others get to know me a bit better. I finally found enough courage to do it when I learned about Hugo. Hugo is an open-source static site generator that uses the command line as its interface.

It is super fast at building and launching a website, just look at that speed!

The tutorial was super helpful and clear.

Hugo made it easy to create and maintain a website by allowing you to choose among several templates and setting everything up from the command line (which I thoroughly enjoy). It also has an amazing mock server that allows you to make changes and immediately see them!

The first task was to choose a good template. This took time as there are a lot of decent templates available, the one I finally settled for is called Hugo-Sustain.

I then decided to host the website on Github as it was the most convenient and clean. The Hugo tutorial will also show you how to do it but if you want to automize the process simply add this script to your root folder and run it.

The script is really straight forward and shows you the individual steps you should take to deploy manually.
